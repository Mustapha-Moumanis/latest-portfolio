Ping Pong Application


Description

This project involves developing a real-time multiplayer Pong game with a modern web-based platform. Key elements include:
Backend: Built with Django and Django REST Framework (DRF) for API development, along with PostgreSQL for secure data handling, user management, JWT-based authentication, and OAuth2 integration.
Frontend: Responsive single-page application using vanilla JavaScript and Bootstrap for styling and multi-browser compatibility.
Gameplay: Local Pong matches, tournaments, and customizable game options with advanced 3D graphics (Three.js).
User Management: Features include secure registration, profile customization, friend system, live chat, and match history.
Security: Implements 2FA, password hashing, HTTPS.

Single Page Application (SPA) :

This project features a dynamic Single Page Application (SPA) built entirely with vanilla JavaScript, showcasing a seamless and responsive user experience. The SPA architecture ensures smooth navigation without full-page reloads by dynamically rendering content based on the current route.

Dockerized Environment :

The project is fully containerized using Docker, ensuring consistent environments and simplified deployment. With a single command (docker-compose up --build), the application autonomously launches all required services, including the Nginx server, Django backend, and PostgreSQL database, providing a seamless and scalable setup.
